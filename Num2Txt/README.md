Number-Text Converter: Feb 6th 2012
-----------------------------------------------------------
This tool converts any given whole number into its textual equivalent and vice-versa.
To use it just call it with number/text as its command-line arguments

Example:

    >>numtxt.py 42
    
    forty two
    
    >>numtxt.py forty two
    
    42
